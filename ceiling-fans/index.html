Most ceiling fan calls are one of two things: a fan upgrade in a room that already has one, or a new fan in a room that doesn't (which often means a new electrical box, new wiring, and a new switch).

Two scenarios

  • Replacing an existing fan or fixture. Easy — typically 60–90 minutes per fan, including disposal of the old unit.
  • Adding a fan where none exists. Requires a fan-rated box, a new circuit run (or pulling a wire from an existing circuit), and possibly a new switch. Typically 2–4 hours per location.

What we always do

  • Use a fan-rated electrical box — regular ceiling boxes can't carry the dynamic load of a fan.
  • Verify the existing wiring is suitable. If it's old aluminum or undersized for fan load, we replace it.
  • Test for proper grounding before final mounting.
  • Balance the blades and verify quiet operation before we leave.
FAQ

Common questions

How long does a fan installation take?

Replacing an existing fan: about an hour. Installing a new fan where there's no existing fixture: 2–4 hours typically.

Can you install a fan in a room with no overhead light?

Yes — we run new wire from a nearby switched circuit, install the fan-rated box, and add a wall switch.

Do you take the old fan?

Yes — we haul off the old unit as part of the installation.

Smart fan or regular?

Up to you. Smart fans are reliable now and worth it if you'd actually use the app or schedule features.
